### 3음으로 코드 보기

보던걸 다르게 보려니까 잘 안되긴 함...그래도 이게 더 빠르다니 별수있나 ㅎㅎ

박치기공룡법으로 계속 해서 익숙해지는 수 밖에 없는듯..

- C - E
- D - F
- E - G
- F - A
- G - B
- A - C
- B - D

근음 = 코드톤 아닌 취급 (어차피 왼손에서 눌러줄꺼기도 하고)
- ex) BbM7인데 탑노트가 Bb이다? -> 3음기반 코드로 Dm만 눌러주면 됨 (Dm7이 아님)

3음보기의 장점 - 코드가 도미넌트7이면 dim7 보게되는데, 이땐 바로 탑노트의 디미니시만 눌러주면 됨 (대신 아래쪽으로 디미니시를 보는 훈련이 필요)
- 코드톤이 아니면? 또 결국 3음의 dim을 봐야 한다...ㅎ 그걸 루트노트에 따라 인버전해야 할 수도 있다...ㅎ 거기에 조표라도 붙으면 진짜그켬

dim7 - 모든 음 간격이 동일함 (모든 음끼리 마이너 간격)
- 신기한 것: Bdmin7 = B, D, F, Ab 이면, Ddim7/Fdim7/(Ab)dim7 전부 구성음이 동일함. 이게 결국 12키밖에 없어서 7-1음 간격도 마이너간격이 되어서임

   - Lydian scale

모드의 종류
- 아이오니안
- 도리안
  - C major 스케일을 D에서 시작한 것과 동일 (b3, b7)
    - Bb 스케일에서의 조표와 동일
    - b3, b7 = 도리안의 `케릭터 노트`. 도리안 스케일 쓴 티를 내고 싶으면 이 케릭터노트를 눌러줘야 함 (ㅡ,.ㅡ)
    - 이거 b3, b7 -> Cm7이 해당 스케일에서 파생된 것
  - 마이너계열: C major 스케일에서 그냥 D를 기반으로 치면 DM이 되는것과 동일. 마찬가지로 프리지안 / 에올리안 / 로크리안이 마이너 계열
- 프리지안
- 리디안
  - C major 스케일을 G에서 시작한 것과 동일 (4#)
- 믹솔-리디안
   - 도미넌트계열 (C major 다이아토닉 코드에서 g는 혼자 도미넌트인 것 처럼)
- 에올리안
- 로크리안


`다이아토닉 코드`
- 특정 으뜸음에서 순서대로 음을 나열한 것
- C major scale의 (7th) 다이아토닉 코드 = 하얀건반만으로 친거 (CM7, Dm7, Em7, FM7, G7, Am7, Bdim7)
- 1, 4, 5도 트라이어드가 중요
   - I: `Tonic`, IV: `Sub-dominant`, V: `Dominant`
