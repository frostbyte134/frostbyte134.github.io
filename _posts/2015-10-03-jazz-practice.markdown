모드의 종류
- 아이오니안
- 도리안
  - C major 스케일을 D에서 시작한 것과 동일 (b3, b7) - D major 스케일에서의 조표와 동일 
    - b3, b7 = 도리안의 `케릭터 노트`. 도리안 스케일 쓴 티를 내고 싶으면 이 케릭터노트를 눌러줘야 함 (ㅡ,.ㅡ)
  - 마이너계열: C major 스케일에서 그냥 D를 기반으로 치면 DM이 되는것과 동일. 마찬가지로 프리지안 / 에올리안 / 로크리안이 마이너 계열
- 프리지안
- 리디안
  - C major 스케일을 G에서 시작한 것과 동일 (4#)
- 믹솔-리디안
   - 도미넌트계열 (C major 다이아토닉 코드에서 g는 혼자 도미넌트인 것 처럼)
- 에올리안
- 로크리안


`다이아토닉 코드`
- 특정 으뜸음에서 순서대로 음을 나열한 것
- C major scale의 (7th) 다이아토닉 코드 = 하얀건반만으로 친거 (CM7, Dm7, Em7, FM7, G7, Am7, Bm7)

코드에서 135음 = 8,10,12음 (+7)
- 멜로디가 코드톤에 없을 때 -> 사실 텐션 주는 게 아니고 그냥 7th 치기
- 멜로디가 코드톤에 있을 때 -> 7th + 근음 생략하고 9th 텐션 주기 연습한 거인듯 (담주에 물어봐야함)

### real book - my romance



<img src="{{site.url}}/images/jazz/my_romance.jpg" width="700" class="center"/> 


무지성 텐션주기 연습
- 멜로디가 코드톤에 없을 시 - 근음 생략하고 근음 밑 7음 (메이저 = 반음, 마이너,도미넌트,플랫파이브 = 온음, 디미니시=온음+반음) 치기
- 멜로디가 코드톤에 있을 시 - 근음 생략하고 9음 치기. 근음기준 메이저/마이너/플랫파이브 - 온음 위, 도미넌트 = 반음, 디미니시 = 3음 (맞나?) 위

2줄 첫마디부터의 클리세 진행 (G코드)은 처음 봄 ㅋㅋ 선생님 말에 따르면 많이 나오는 듯
- 두번째의 G-(maj7)은 정작 G는 치지 말라고 하심. 왠지모르겠는데 너무 많이 치게 되서? 어차피 왼손에서 근음은 눌러주기때문? 인 듯
