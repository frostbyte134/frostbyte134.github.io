#### HTTP 소개
- HTTP = 크게 Request/Response로 나뉨
- 크롬의 우클릭-검사 (inspect) - Network탭 : 크롬의 네트워크 모니터링 툴
  - `<img src="asd.jpg" ... >` 를 브라우저가 읽을 시, `asd.jpg`를 요청하기 위한 HTTP통신이 이루어짐
  - 각 HTTP 통신 (request-response pair, network탭의 1줄) 에 대해 header를 읽을 수 있음
