`multiset`으로 풀었어도 되고...set도 index붙이면 unique해서 됐을듯. 담부턴 multiset 쓰자  
```c++
while (pq.size() && pq.begin()->first < p)pq.erase(pq.begin());
while (pq.size() > K) {
   auto it = prev(pq.end());
   pq.erase(it);
}
```
